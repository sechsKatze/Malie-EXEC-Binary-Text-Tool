Malie-EXEC-Binary-Text-Tool
======
This Python script extracts script data from exec.dat (EXEC), the text resource file used by the Malie System engine, and converts it into JSON format for editing and analysis.

Usage
======
1. Place this script and the exec.dat (EXEC) file in the same directory. Then open a CMD console window.
(The quickest way is to type cmd in the directory path bar and press Enter.)
2. In the console, run the following command: "python exec_tool.py disassemble EXEC output.json"
